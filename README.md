# applicant-research-tool

I want to make a job applicant research tool. It should take an applicants name then call an api to research that applicant. 10 seconds later it should open up an information panel about the results of the research.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/applicant-research-tool.git
cd applicant-research-tool
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
